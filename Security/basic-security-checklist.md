# Basic Security Checklist

## Overview

Maintaining a secure computer is a fundamental responsibility for both IT Support professionals and end users. This checklist outlines essential security practices to reduce security risks and protect organizational data.

---

## Account Security

- Use a strong and unique password.
- Enable Multi-Factor Authentication (MFA) whenever available.
- Never share passwords with others.
- Lock your computer when leaving your desk.
- Change passwords immediately if compromise is suspected.

---

## System Security

Verify that:

- Windows is up to date.
- Automatic updates are enabled.
- Windows Firewall is enabled.
- Microsoft Defender Antivirus is running.
- Real-time protection is enabled.

---

## Software Security

- Install software only from trusted sources.
- Remove unused applications.
- Keep installed software updated.
- Avoid using unauthorized or pirated software.

---

## Email Security

Before opening an email:

- Verify the sender.
- Check for suspicious links.
- Do not open unexpected attachments.
- Report phishing emails to the IT department.

---

## Internet Security

- Visit only trusted websites.
- Verify HTTPS before entering sensitive information.
- Avoid downloading unknown files.
- Do not disable browser security warnings.

---

## Device Security

- Lock your computer when unattended.
- Do not connect unknown USB devices.
- Encrypt sensitive data if required.
- Secure laptops during travel.

---

## Backup

Verify that:

- Important files are backed up regularly.
- Backup files are tested periodically.
- Recovery procedures are documented.

---

## Physical Security

- Secure access to workstations.
- Prevent unauthorized access to office equipment.
- Keep confidential documents protected.
- Report lost or stolen devices immediately.

---

## Security Incident Response

Report immediately if you notice:

- Suspicious login attempts.
- Unknown software installations.
- Malware infections.
- Lost or stolen devices.
- Unauthorized access to company data.

---

## Daily Security Checklist

Before starting work:

- Verify antivirus is running.
- Check Windows Update status.
- Confirm firewall is enabled.

Before leaving work:

- Lock or shut down your computer.
- Close sensitive applications.
- Secure confidential documents.

---

## Best Practices

- Follow company security policies.
- Complete security awareness training.
- Use approved software only.
- Report suspicious activity immediately.
- Keep all systems updated.
