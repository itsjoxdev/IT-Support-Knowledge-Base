# Keyboard & Mouse Troubleshooting

## Overview

Keyboard and mouse issues can significantly impact user productivity. This guide provides a structured approach to diagnosing and resolving common input device problems on Windows systems.

---

## Symptoms

Users may experience:

- Keyboard not responding
- Mouse not moving
- Delayed keyboard input
- Cursor freezing
- Some keys not working
- Wireless keyboard or mouse disconnecting
- USB device not recognized

---

## Possible Causes

- Loose USB connection
- Low battery (wireless devices)
- Faulty USB port
- Corrupted or outdated drivers
- Hardware failure
- Bluetooth connection issues
- Windows configuration problems

---

## Troubleshooting Methodology

### Step 1 – Check Physical Connections

For wired devices:

- Verify the USB cable is securely connected.
- Try another USB port.

For wireless devices:

- Ensure the device is powered on.
- Check battery level.

---

### Step 2 – Restart the Computer

Restart Windows and test the device again.

---

### Step 3 – Test Another USB Port

Connect the keyboard or mouse to another USB port.

If available, test the device on another computer.

---

### Step 4 – Check Device Manager

Open:

```
Device Manager
```

Verify:

- No warning icons are displayed.
- The device is detected correctly.

Update or reinstall the driver if needed.

---

### Step 5 – Test Bluetooth Connection

For Bluetooth devices:

- Turn Bluetooth off and on.
- Remove the device.
- Pair it again.

---

### Step 6 – Run Windows Troubleshooter

Open:

```
Settings → System → Troubleshoot
```

Run the Hardware Troubleshooter if available.

---

### Step 7 – Update Drivers

Use:

- Windows Update
- Device Manager
- Manufacturer's official driver

---

## Useful Commands

```cmd
devmgmt.msc
```

Open Device Manager.

```cmd
control
```

Open Control Panel.

```cmd
ms-settings:bluetooth
```

Open Bluetooth Settings.

---

## Resolution

Most keyboard and mouse issues can be resolved by:

- Reconnecting the device
- Replacing batteries
- Updating drivers
- Repairing the Bluetooth connection
- Using another USB port
- Replacing faulty hardware

---

## Prevention

- Keep device drivers updated.
- Replace batteries before they are fully depleted.
- Avoid damaging USB cables.
- Clean the keyboard and mouse regularly.
- Use high-quality peripherals.
