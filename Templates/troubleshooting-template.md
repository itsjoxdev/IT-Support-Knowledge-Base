# Troubleshooting Template

## Overview

Use this template to document new IT support issues in a consistent and professional format. Following a standardized structure improves troubleshooting efficiency, documentation quality, and knowledge sharing.

---

# Issue Title

Provide a short and descriptive title.

Example:

```
Unable to Connect to VPN
```

---

## Overview

Briefly describe the issue.

Example:

> Users are unable to establish a VPN connection while working remotely.

---

## Symptoms

List the observed symptoms.

Example:

- Connection timeout
- Authentication failure
- VPN disconnects frequently

---

## Possible Causes

Identify potential root causes.

Example:

- Incorrect credentials
- Internet connectivity issues
- VPN server outage
- Expired user certificate
- Firewall restrictions

---

## Useful Commands

## Resolution

Explain how the issue was resolved.

Example:

- Updated network settings
- Restarted required services
- Reinstalled the application
- Corrected configuration

---

## Prevention

List recommendations to prevent the issue from happening again.

Example:

- Keep software updated.
- Follow company security policies.
- Verify network configuration regularly.
- Maintain current device drivers.

---

## Escalation

Escalate the issue if:

- The problem persists after completing all troubleshooting steps.
- Multiple users are affected.
- Hardware replacement is required.
- A security incident is suspected.
